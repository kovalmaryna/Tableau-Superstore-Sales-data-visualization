# Tableau Project: Superstore Sales Data Visualization
Final project of the Tableau block in the Data Analyst courses from GoIT

## Project Overview
This project focused on building interactive dashboards in Tableau using the Superstore Sales dataset.  
The goal was to analyze sales and profitability, study delivery times across the USA, and enhance dashboards with interactive elements.

## Business Value & Insights
- **Sales Trends:** Built time series of sales and profits (2019–2023) to identify growth and decline periods.  
- **Profitability Analysis:** Compared profitability across subcategories and shipping methods, highlighting weak and strong product groups.  
- **Delivery Performance:** Visualized delivery times by shipping mode and by state in the USA.  
- **Interactive Dashboards:** Integrated filters, actions, and parameters to improve user experience and dynamic analysis.  

## Technical Stack
- **BI Tool:** Tableau Public  
- **Dataset:** Superstore Sales  

## Tasks
### 1. Sales and Profit Time Series
- Built a dual-axis line chart for monthly aggregated sales and profit.  
- Configured filters for Ship Mode and Segment with "Apply" buttons.  

### 2. Profitability by Subcategories and Shipping Methods
- Created a heatmap with Category/Sub-category vs Ship Mode.  
- Calculated relative profit (%) and applied a two-color scale with zero as neutral.  
- Added subtotals for rows and columns.  

### 3. Delivery Time Analysis
- Average delivery days by Ship Mode.  
- Orders grouped by delivery duration (0–8 days).  
- Average delivery days by state visualized on a choropleth map.  
- Combined all three visuals into one dashboard with filters (Order Date, Segment, Ship Mode).  

### 4. Sales Funnel Interactivity (Extended Task)
- Added actions and parameters to connect funnel steps with delivery time metrics.  
- Enabled dynamic filtering based on user interaction with funnel stages.  

## Results
- **Dashboards:** Published in Tableau Public with interactive filters and parameters.  
- **Visualizations:** Sales trends, profitability heatmap, delivery time analysis, and interactive funnel.  
- **User Experience:** Improved dashboard usability with dynamic interactions.
